# JD Web Scraper

## Data scraped

* ID
* Nickname
* Bought Time
* Comment Time
* No. of Likes
* No. of Replies
* Phone Color
* Phone Memory
* Sales Set
* Rating (Good = 4-5, Middle = 2-3, Bad = 1)
* Content
* Source url

## Logic

* Analysing network activities of JD Product comment summary

* Using requests to acquire json data
